💻 How to Run

The application is ready to use! You have two options:

Option 1: Direct Browser (Easiest)

Simply open index.html in your web browser.

Option 2: Local Server (Recommended)

For the best experience, run a local server:

# Windows: Python 3 python -m http.server 8000 # Mac/Linux: Python 3 python3 -m http.server 8000 # Using Node.js npx http-server # Then open: http://localhost:8000

🧪 Try These Test Scenarios

  1. Legitimate Job: "Senior Software Engineer at TechCorp, 10+ years experience required, $150k-200k, Full details at careers.techcorp.com"
  2. Suspicious Job: "Work from home, no experience needed, earn $5000 daily! Send WhatsApp to contact"
  3. Clear Scam: "Pay $500 registration fee to start, unlimited income, urgent hiring!"
